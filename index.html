2 Processing the HandMeThat tasks with GPT3.5

2.1 Open gesture set

2.1.1 move

  1. Prompt

    How would you communicate, using only hand gestures, to persom X that you want him to move from location Z to location W?

  2. Completions
    • I would point in the direction of location W and make a walking gesture with my hands, indicating my desire for person X to move from location Z to location W.
    • First, point to the person and then point in the direction of location W. Then, make walking motions with your hands and follow it with a pointing motion in the direction of location W. Finally, make a nodding gesture to indicate that you want the person to move to location W.

2.1.2 pick-up-at-loc

  1. Prompt

    How would you communicate, using only hand gestures, to person X that you want him to pick up object Y at location W.

  2. Completion

    First, point at person X. Then, hold your hand up in the air with your fingers pointing up like you’re holding something. Then, point in the direction of location W. Finally, hold your hand up in the air again with your fingers pointing up like you’re holding something and move your hand back towards person X to indicate that you want them to pick up object Y.

2.1.3 put-inside-rec-at-loc

  1. Prompt

    How would you communicate, using only hand gestures, to person X that you want him to put object Y into container W, which is at location Z?

  2. Completion

    First, you would point to object Y and then make a pushing motion towards location Z. Then, you would point at container W and motion with both hands as though you were lifting an object and placing it inside. Lastly, you would point at container W to indicate where the object should be placed.

2.2 Closed gesture set

2.2.1 pick-up-at-loc

  1. Prompt
    • How would you express the intention below if you could use only the following hand gestures without words:
    • pointing to something,
    • waving at something,
    • making a circular motion.
    • moving your hand vertically.
    • moving your hand horizontally.

    Intention: You want person X to put object Y into container W, which is at location Z.

    1. Point to person X.
    2. Point to object Y.
    3. Move hand hotizontally in the direction of W.

    Intention: You want person X to pick up object Y at location W.

  2. Completion

    Point to person X.

    1. Point to location W.
    2. Make a circular motion with your hand.
